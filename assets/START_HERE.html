❓ Common Questions

Q: Where do I put these files?
A: Copy the entire pixel-assets/ folder into your game directory next to index.html

Q: How do I use them?
A: Load with new Image() and draw with ctx.drawImage(). See INTEGRATION_GUIDE.md for examples.

Q: Can I modify the sprites?
A: Yes! Edit PNG files in any image editor, or run create-premium-assets.py to regenerate.

Q: Do I need more assets?
A: Check ASSET_SOURCES.md for links to free assets on OpenGameArt, itch.io, and CraftPix.
